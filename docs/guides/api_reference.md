# APIリファレンス

MochiRAGのバックエンドはFastAPIを利用して構築されており、OpenAPI仕様に準拠した対話的なAPIドキュメントを自動生成します。

手動で管理される静的なドキュメントは古くなる可能性があるため、APIの仕様については、常にこの自動生成されたドキュメントを正として参照してください。

## APIドキュメントへのアクセス

1.  [開発者セットアップガイド](./developer_setup_guide.md)に従って、バックエンドサーバーをローカルで起動します。
    ```bash
    uvicorn backend.main:app --reload --port 8000
    ```

2.  Webブラウザで以下のURLにアクセスします。

    **http://localhost:8000/docs**

上記URLで、すべてのAPIエンドポイント、リクエスト/レスポンスのスキーマ、および認証要件を対話的に確認し、直接APIを試すことができます。
